# 🌤️ WeatherTracker

Aplicação web de previsão do tempo em tempo real, desenvolvida com HTML, CSS e JavaScript puro.

## 📸 Demonstração

> Pesquise qualquer cidade do mundo e veja o clima na hora!

## ✨ Funcionalidades

- 🌡️ Temperatura atual em °C e °F
- 🥵 Sensação térmica
- 💧 Umidade do ar
- 💨 Velocidade do vento
- 🌤️ Descrição do clima com emojis dinâmicos
- 🕐 Data e hora em tempo real
- 🔵 Histórico das últimas cidades pesquisadas
- 📱 Layout responsivo para celular

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- [OpenWeatherMap API](https://openweathermap.org/)

## 🚀 Como usar

1. Clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Digite o nome de uma cidade e clique em **Buscar**
